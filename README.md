# Black Heritage Friesian Stud

## Учебный веб-проект для производственной практики

Проект выполнен в рамках профессионального модуля **ПМ.09 «Проектирование, разработка и оптимизация веб-приложений»**.

**Black Heritage Friesian Stud** — это адаптивный сайт для племенного завода фризских лошадей. Проект разработан как демонстрационное веб-приложение с клиентской частью, серверной обработкой формы, подключением к базе данных и простой административной страницей для просмотра заявок.

## Идея проекта

Идея проекта — создать премиальный сайт для элитного племенного хозяйства, специализирующегося на фризских лошадях.

Визуальная концепция построена на:

* тёмной цветовой палитре;
* золотых акцентах;
* крупных фотографиях лошадей;
* спокойной презентационной подаче;
* ощущении закрытого премиального клуба.

Сайт позволяет пользователю познакомиться с направлением работы завода, посмотреть лошадей на продажу, изучить производителей и оставить заявку на частный просмотр.

---

## Назначение сайта

Сайт предназначен для:

* презентации племенного завода;
* демонстрации фризских лошадей на продажу;
* представления производителей хозяйства;
* описания услуг и преимуществ;
* сбора заявок на частный просмотр и подбор лошади;
* просмотра заявок через учебную административную страницу.

## Основные разделы сайта

На сайте реализованы следующие разделы:

* **Главный экран** — презентация Black Heritage Friesian Stud;

* **Административная страница** — просмотр заявок.

---

## Особенности раздела «На продажу»

Раздел **«На продажу»** выполнен как премиальный каталог лошадей.

Пользователь может выбрать лошадь из списка, после чего на странице меняются:

* фотографии выбранной лошади;
* характеристики;
* описание;
* кнопка заявки на просмотр.

Для каждой лошади используется несколько фотографий. Изображения отображаются в крупном формате, а оформление раздела выполнено в тёмной стилистике с золотыми акцентами.

---

## Форма заявки

На сайте реализована форма заявки на частный просмотр и подбор лошади.

Форма собирает следующие данные:

* имя;
* телефон;
* email;
* интересующая лошадь;
* цель подбора;
* формат просмотра;
* удобная дата;
* комментарий;
* согласие на обработку персональных данных.

После отправки данные проходят проверку и сохраняются в базу данных.

## Административная страница

Для просмотра заявок создана учебная административная страница:

```text
admin.php
```

Она получает данные из таблицы `viewing_requests` и выводит их в виде таблицы.

На странице отображаются:

* имя;
* телефон;
* email;
* интересующая лошадь;
* цель подбора;
* формат просмотра;
* удобная дата;
* комментарий;
* дата создания заявки.

В учебном проекте административная страница не закрыта авторизацией. В реальном проекте такую страницу обязательно нужно защищать логином и паролем.

## Размещение на хостинге

Проект размещён на бесплатном PHP-хостинге **InfinityFree**.

Для размещения были выполнены следующие действия:

1. Создан поддомен `blackheritage.xo.je`.
2. Файлы проекта загружены в папку `htdocs`.
3. Создана база данных MySQL.
4. Через phpMyAdmin выполнен SQL-скрипт создания таблицы `viewing_requests`.
5. В файле `includes/db.php` указаны данные подключения к базе на хостинге.
6. Проверена отправка формы.
7. Проверен вывод заявки на странице `admin.php`.

## Безопасность

В проекте реализованы базовые меры безопасности:

* данные формы очищаются перед обработкой;
* email проверяется на корректность;
* значения списков проверяются на допустимые варианты;
* дата не может быть прошедшей;
* запись в базу выполняется через подготовленные SQL-запросы;
* данные в админке выводятся через `htmlspecialchars`;
* реальные данные подключения к базе не публикуются в GitHub.
