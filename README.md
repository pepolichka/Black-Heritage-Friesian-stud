Black Heritage Friesian Stud

Учебный веб-проект для производственной практики по профессиональному модулю ПМ.09 «Проектирование, разработка и оптимизация веб-приложений».

Проект представляет собой адаптивный сайт для племенного завода фризских лошадей Black Heritage Friesian Stud. Сайт разработан как демонстрационное веб-приложение с клиентской частью, серверной обработкой формы, подключением к базе данных и простой административной страницей для просмотра заявок.

Идея проекта

Идея проекта — создать премиальный сайт для элитного племенного хозяйства, специализирующегося на фризских лошадях. Визуальная концепция построена на тёмной цветовой палитре, золотых акцентах, крупных фотографиях лошадей и спокойной презентационной подаче.

Сайт должен не просто показывать информацию о хозяйстве, а создавать ощущение закрытого премиального клуба, где пользователь может познакомиться с направлением работы завода, посмотреть лошадей на продажу, изучить производителей и оставить заявку на частный просмотр.

Назначение сайта

Сайт предназначен для:

презентации племенного завода;
демонстрации фризских лошадей на продажу;
представления производителей хозяйства;
описания услуг и преимуществ;
сбора заявок на частный просмотр и подбор лошади;
просмотра заявок через учебную административную страницу.
Основные разделы сайта

На сайте реализованы следующие разделы:

главный экран с презентацией Black Heritage Friesian Stud;
раздел «О нас» с описанием хозяйства;
раздел «На продажу» с интерактивным каталогом лошадей;
раздел «Наши производители»;
раздел с молодняком;
раздел услуг;
блок достижений и преимуществ;
форма заявки на частный просмотр;
контактный блок и подвал сайта;
административная страница для просмотра заявок.
Особенности раздела «На продажу»

Раздел «На продажу» выполнен как премиальный каталог лошадей. Пользователь может выбрать лошадь из списка, после чего меняются:

фотографии выбранной лошади;
характеристики;
описание;
кнопка заявки на просмотр.

Для каждой лошади используется несколько фотографий. Изображения отображаются в крупном формате, а сама подача раздела выполнена в тёмной стилистике с золотыми акцентами.

Форма заявки

На сайте реализована форма заявки на частный просмотр и подбор лошади. Форма собирает следующие данные:

имя;
телефон;
email;
интересующая лошадь;
цель подбора;
формат просмотра;
удобная дата;
комментарий;
согласие на обработку персональных данных.

После отправки данные проходят проверку и сохраняются в базу данных.

Административная страница

Для просмотра заявок создана учебная административная страница:

admin.php

Она получает данные из таблицы viewing_requests и выводит их в виде таблицы. На странице отображаются имя, телефон, email, интересующая лошадь, цель подбора, формат просмотра, удобная дата, комментарий и дата создания заявки.

В учебном проекте административная страница не закрыта авторизацией. В реальном проекте такую страницу обязательно нужно защищать логином и паролем.
